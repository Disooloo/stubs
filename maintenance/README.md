# Заглушка «Техническое обслуживание»

Статическая HTML-страница, которую можно отдавать пользователям во время плановых работ, обновлений или аварийного простоя сервиса. Тексты на русском, адаптивная вёрстка, акцент на понятном сообщении и контакте поддержки.

![Превью заглушки технического обслуживания](maintenance/maintenance.png)

## Содержимое

| Файл | Назначение |
|------|------------|
| `maintenance/maintenance.html` | Страница-заглушка (открыть в браузере или отдать с веб-сервера) |
| `maintenance/maintenance.png` | Иллюстрация для README и опционально для блока на странице |

## Как использовать

1. Скопируйте папку `maintenance` на хостинг или в корень статики.
2. В настройках балансировщика, CDN или веб-сервера укажите редирект или подстановку этой страницы на время работ.
3. При необходимости замените почту и текст в `maintenance.html` под ваш продукт.

Локальный просмотр: откройте `maintenance/maintenance.html` в браузере (двойной щелчок или «Открыть с помощью»).

## Заметки

- Шрифты подключаются с Google Fonts; при полностью офлайн-сценарии замените их на системные стеки в CSS.
- Картинка `maintenance.png` дублируется на странице в блоке-герое для единообразия с документацией.
